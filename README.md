# Chai Culture — Brew the Royal Tradition

A premium **Shopify-style “Coming Soon” storefront UI** built for **Chai Culture**, inspired by **royal Indian heritage**, warmth, and authenticity.  
This project showcases an elegant launch-ready e-commerce experience with premium styling, product showcase, and a coming soon signup flow.

---

## Project Overview

Chai Culture is a luxury chai brand designed around:
- **Royal heritage**
- **Warm & elegant visual identity**
- **Premium launch experience**
- **Shopify-like storefront structure**

This project includes a **Coming Soon homepage**, product showcase, shop page, product details, and contact/FAQ.

## Key Features

- Luxury **Coming Soon launch page**
- Premium **Announcement bar** (“Coming Soon” banner)
- **Notify Me** email capture (frontend)
- Product grid with:
  - badges
  - price
  - quick view modal
- Product detail pages with:
  - tasting notes
  - brew time
  - ingredients section
- Shop page with:
  - filter + sort UI (Shopify-like)
- About page with royal brand storytelling
- Contact page with:
  - form
  - FAQ accordion
- Fully responsive & mobile-first
- Smooth animations + premium hover effects
